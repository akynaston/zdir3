2. Run the program and login with the following information:

* Host name: 71.163.15.59 (will be [ftp.trivir.com](ftp://ftp.trivir.com/))
* User name: trivirvm
* Password: rKmqnFaF
* Port number 2722
* Protocol: Use SFTP with SCP fallback
